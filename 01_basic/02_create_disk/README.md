# Terraform for さくらのクラウド ハンズオン

## 基本編02: ディスクの作成〜サーバとの接続

ディスクを作成し、サーバと接続します。

![基本編02](../images/02.png "基本編02")

### 解説

ディスクを作成し、サーバと接続します。
この段階ではまだディスクにOSがインストールされていません。

## コマンド

* `terraform plan` … 確認
* `terraform apply` … 反映
* `terraform show` … 詳細情報の表示
* `terraform destroy` … 環境の破棄
